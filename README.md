# DIU Smart Campus 🎓📱

**DIU Smart Campus** is a modern, high-performance mobile application built for students at Daffodil International University. It centralizes essential campus resources—routines, notices, bus schedules, and exam info—into a single, seamless platform.

## 🚀 The Problem & Solution
University life often involves navigating multiple websites and portals to find simple information. This app bridges that gap by providing a unified hub, saving students significant time and effort.

## ✨ Key Features
*   **Centralized Student Hub**: Integrated access to student email and official notices.
*   **Real-time Routines**: Class schedules fetched dynamically, organized by batch.
*   **Exam Information**: Instant access to course-specific exam schedules and seat plans.
*   **Bus Tracking**: Up-to-date university bus schedules for efficient commuting.
*   **Professional UI/UX**: A sharp, clean "Dark Mode" aesthetic using a custom palette for maximum visibility.
*   **Dedicated About & Support**: Independent screens for app information and troubleshooting help.

## 🛠️ Technical Stack
*   **Frontend**: Flutter (Dart)
*   **Backend**: Firebase (Cloud Firestore & Authentication)
*   **Architecture**: Optimized for scalability and clean state management.
*   **Deployment**: Production-ready APK built and tested on physical Android devices.
